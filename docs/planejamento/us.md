# Histórias de Usuário

Histórias de Usuário são descrições curtas de funcionalidades, contadas a partir da perspectiva do usuário final. Elas focam no valor que a funcionalidade entrega, e não nos detalhes técnicos, servindo como base para a comunicação e o desenvolvimento centrado no usuário.
